💊 RHU PHARMACY NCD PATIENT SYSTEM

A web-based application designed to manage pharmacy patient records, maintenance medicines, and vitamins. This system improves workflow efficiency for healthcare staff by providing a structured and user-friendly interface for handling patient data.

🚀 Features
🧑‍⚕️ Patient Management (Create, Read, Update, Delete)
💊 Maintenance Medicine Tracking
🧾 Vitamin Records Management
🔍 Search and Filter Patients
✏️ Edit Patient Information via Modal
⚡ Real-time UI updates
📡 RESTful API integration

🛠️ Tech Stack
Frontend: Next.js (App Router), TypeScript
Styling: Tailwind CSS
Backend: Next.js API Routes (RESTful APIs)
HTTP Client: Axios
Database: MongoDB Atlas
ODM: Mongoose
